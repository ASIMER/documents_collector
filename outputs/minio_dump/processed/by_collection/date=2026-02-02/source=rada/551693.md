	<h3>Перевірка</h3>
	<p>Натисніть, будь-ласка, на це <a href="/laws/show/4683-20?test=YOBMfttwf6EIa2yMZitFkFUdHXpwYsEsGcuDpz6" target=_top><b>посилання</b></a>, якщо браузер не переадресував на документ автоматично.</p>
	<p>Документи потрібно відкривати по одному, а не кілька зразу. Будьте уважні - багаторазове натискання на одне те саме посилання блокує доступ на деякий час! Кількість переадресацій можна зменшити, якщо у браузері працюють кукі.</p>
	<p>Цей захист зроблено для перевірки на автоматичні запити для <a href="/laws/main/rules">запобігання хакерським атакам</a>.</p>
