КАБІНЕТ МІНІСТРІВ УКРАЇНИ
ПОСТАНОВА від 6 серпня 2003 р. N 1211 Київ
Деякі питання реекспорту товарів
{ Із змінами, внесеними згідно з Постановами КМ N 875 ( 875-2021-п ) від 18.08.2021 N 1653 ( 1653-2025-п ) від 10.12.2025 }
З метою виконання міжнародних договорів України та відповідно до статті 197 Митного кодексу України ( 92-15 ) Кабінет Міністрів України постановляє:
1. Уповноважити Міністерство економіки, довкілля та сільського господарства видавати дозволи на реекспорт товарів, переліки яких оприлюднюються Міністерством: { Абзац перший пункту 1 із змінами, внесеними згідно з Постановами КМ N 875 ( 875-2021-п ) від 18.08.2021, N 1653 ( 1653-2025-п ) від 10.12.2025 }
походженням з України, що здійснюється з території інших країн відповідно до міжнародних договорів України;
походженням з інших країн, що здійснюється за межі митної території України.
2. Державній митній службі затвердити до 1 січня 2004 р. порядок застосування митних процедур до товарів, що переміщуються через митний кордон України в режимі реекспорту ( z0983-03 ).
3. Дія цієї постанови не поширюється на товари, реекспорт яких здійснюється відповідно до Указу Президента України від 13 лютого 1998 р. N 117 ( 117/98 ) "Про Положення про державний експортний контроль в Україні".
4. Ця постанова набирає чинності з 1 січня 2004 року.
Прем'єр-міністр України В.ЯНУКОВИЧ
